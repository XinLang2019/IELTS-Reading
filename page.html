Title 1: Introduction to Web Development

1.1 What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages.

1.2 What is CSS?

CSS (Cascading Style Sheets) is used to style HTML elements.

Title 2: JavaScript Basics

2.1 What is JavaScript?

JavaScript is a programming language that allows you to add interactivity to web pages.

2.2 Variables and Data Types

JavaScript variables can store different data types, such as numbers, strings, and objects.
